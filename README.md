# üõí BlinkIT Grocery Sales Analysis Dashboard

## üëã Quick Overview

This project is a detailed **Grocery and Retail Sales Dashboard** built to analyze the performance of the BlinkIT business. I put this together to help quickly track sales, understand product popularity, and see which store types are generating the most revenue.

The goal is simple: **Turn raw sales data into smart, easy-to-digest business decisions.**

***

## üöÄ Key Insights & Features

The analysis covers a wide range of retail factors, using clear visualizations to highlight trends and drivers of success.

| Category | Key Metrics & Analysis |
| :--- | :--- |
| **Core Performance (KPIs)** | Total Sales, Average Sales, Number of Items Sold, and the Average Customer Rating. |
| **Product Segmentation** | Breakdown of sales by **Item Type** (e.g., Snack Foods, Fruits and Vegetables, Soft Drinks, Dairy, Household). |
| **Fat Content Impact** | Analysis of Total Sales segmented by **Item Fat Content** (Low Fat vs. Regular). |
| **Outlet Performance** | Sales performance tracked by **Outlet Establishment Year** (e.g., 2012, 2016, 2018), **Outlet Location Type**, **Outlet Size**, and **Outlet Type**. |
| **Customer Feedback** | Analysis of sales volume correlated with customer **Rating**. |

## üìÇ Getting Started

Want to open the data files and review the analysis structure?

### What's Included

The repository contains all the necessary data and metric sheets for the analysis:

* `BlinkIT.xlsb .xlsx - BlinkIT Grocery Data.csv`: The raw transaction-level dataset.
* `BlinkIT.xlsb .xlsx - Sheets Design.csv`: Contains the calculated KPI's and segmented sales tables (like Sales by Fat Content and Item Type).
* `BlinkIT.xlsb .xlsx - Dashboard.csv`: A representation of the final dashboard layout.

### How to Use

1.  **Download/Clone:** Get a copy of the repository files.
2.  **Open the Data:** You can open any of the CSV files in Excel or a spreadsheet program to review the raw data and the pre-calculated metrics.
3.  **View the Dashboard:** If you have the original Excel file (`.xlsb`), you can open it to see the fully interactive visualization. (The included CSVs are exports of the data and sheet designs).
